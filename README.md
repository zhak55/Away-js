# Away.js
Away.js is simple lib to manage your links in your app or site

### Get started 

```
<script type="text/javascript"> 
 $(function(){
   $("a").away({
    template : "You are going to {{link}}. Are you sure?"
   })
 });
</script>

<!-- in html -->
<a href="https://www.gmail.com/">Go to Mail</a>
```
